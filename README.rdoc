==About

This is a road guide mapless, It is a webapp system for Dwesa (South Africa) inhabitants that provides a simple way of offering/requesting rides and publishing road problems to its users.

This project was presented during the 1st South Africa Hackathon Challenge, hosted in Trento - Italy. 

The code can be tried out in Heroku: http://dwesa.herokuapp.com/
